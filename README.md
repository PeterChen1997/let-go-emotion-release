# Let Go · 情绪释放练习

一个面向 GitHub Pages 的 iOS 风格情绪自助练习 Web App。

## 已实现功能
- 快速选择主情绪
- 选择更细的二级情绪
- 选择身体感受
- 记录一句话触发事件
- 记录练习前后强度、清晰度、身体放松度
- 保存历史记录到 localStorage
- 非医疗边界说明

## 已支持的情绪分类

### 主情绪分组
1. 焦虑
2. 愤怒
3. 悲伤
4. 羞耻
5. 恐惧
6. 受伤
7. 麻木
8. 混乱
9. 喜悦
10. 平静

### 二级情绪词
- 焦虑：紧张、不安、心慌、反复担心、害怕出错、压力很大、停不下来、预感不好、坐立不安
- 愤怒：生气、烦躁、火大、不爽、被冒犯、想反击、不甘心、受不了、想发火
- 悲伤：难过、失落、沮丧、想哭、空落落、被抛下、无力、低沉、泄气
- 羞耻：羞耻、尴尬、自责、内疚、觉得自己不够好、丢脸、后悔、想躲起来
- 恐惧：害怕、怕失控、怕失败、怕被拒绝、怕冲突、怕麻烦变大、怕没人支持、怕出事
- 受伤：委屈、心寒、被忽视、被误解、失望、受伤、被否定、不被看见、心里堵
- 麻木：麻木、空白、没感觉、断联感、呆住、提不起劲、迟钝、像被冻住
- 混乱：混乱、纠结、拿不准、脑子乱、想太多、看不清、矛盾、被卡住
- 喜悦：轻松、开心、被支持、释然、有希望、满足、感谢、被理解
- 平静：平静、稳定、松开了、安心、踏实、清楚、缓下来了、能呼吸了

### 身体感受
胸口堵住、喉咙发紧、胃里发沉、肩膀很紧、头胀、心跳快、想哭、想逃开、身体发麻
